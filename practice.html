CSS Basics

CSS (Cascading Style Sheets) is a styling language used to control the layout and appearance of web pages written in HTML or XML. A CSS stylesheet is a file or a section of code that contains rules and properties that dictate how elements on a web page should bedisplayed

CSS uses

Think of it like a recipe book for your website's visual design! You can use CSS to

  • Change colors, fonts, and sizes
  • Add borders, margins, and padding
  • Create layouts and position element
  • Add animations and effects
